# restaurant-analyse-duesseldorf
Standort- und Wettbewerbsanalyse für ein indisches Restaurant in Düsseldorf

# Projektziel:

Als Wirtschaftsinformatik-Studentin habe ich eine vollständige Marktanalyse durchgeführt, um folgende Fragen datenbasiert zu beantworten:
-  **Wer ist die Zielgruppe?** Wo leben in Düsseldorf die meisten Menschen mit internationalem Hintergrund?
- **Wann gehen Menschen essen?** Analyse der Stoßzeiten der Konkurrenz
- **Wo ist die Konkurrenz?** Welche Stadtteile sind bereits gesättigt?

# Tech Stack:
tools - verwendung:
PostgreSQL - Datenspeicherung & SQL-Abfragen
PgAdmin 4 - Datenbank-Management
Power BI - Dashboard & Visualisierung
OpenStreetMap -  Geodaten der Konkurrenz
Open Data Düsseldorf - Bevölkerungsdaten

# Ergebnisse:

# Zielgruppe:
- **Friedrichstadt** hat mit 437 die meisten indischen Staatsangehörigen
- **Stadtmitte** hat den höchsten prozentualen Anteil (2,72%)

# Stoßzeiten
- **Samstag** ist der stärkste Tag, 4 von 5 Restaurants haben Stoßzeit
- Hauptzeit: **ab 18 Uhr**

#  Konkurrenz
- 3 von 5 indischen Restaurants konzentrieren sich in der **Stadtmitte**
- **Nord** hat nur 2 Konkurrenten — weniger gesättigter Markt
